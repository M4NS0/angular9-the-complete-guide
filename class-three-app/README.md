# Recipes and Shopping List #

## Adding components to the Project ##
```sh
    ng g c component-name/nested-component-name
```

### Structure: ###

> Header Component:
```sh
   /header
```

> Recipes Component:
```sh
  /recipes/
  /recipes/recipes-detail
  /recipes/recipes-list
  /recipes/recipes-list/recipe-item
```

> Shopping list Component
```sh
  /shopping list
```
